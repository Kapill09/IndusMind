# INDUS MIND

An Agentic AI Platform for Industrial Knowledge & Decision Intelligence

## Objective

INDUS MIND helps industrial engineers investigate machine failures by retrieving information from manuals, SOPs, maintenance logs, inspection reports, and sensor data using a multi-agent AI architecture.

## Tech Stack

- Python
- FastAPI
- LangGraph
- LangChain
- ChromaDB
- OpenAI
- React
- TailwindCSS
